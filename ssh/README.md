# SSH 配置目录

本目录原用于存放 SSH 私钥，现已改为**密码登录**模式，不再需要密钥文件。

## 部署方式

`deploy.py` 使用密码登录服务器：

1. **环境变量（推荐）**：
   ```bash
   export SSH_PASSWORD="你的服务器密码"
   ```

2. **直接修改 `deploy.py`**（不推荐）：
   编辑 `deploy.py` 顶部的 `SSH_PASSWORD` 默认值。
